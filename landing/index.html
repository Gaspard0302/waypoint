How It Works

From script tag to
AI navigation in minutes.

01

Paste one line

Drop the script tag into your <head>. Works on any site — React, Next.js, Vue, plain HTML.

<script
  src="https://cdn.waypoint.ai/v1.js"
  data-key="your-api-key"
  defer
></script>
02

We index your site

Our crawler maps every page, button, and link — building a compact action graph. Re-indexes automatically on deploy.

Get your API key → 
<!-- Add to your <head> -->
<script
  src="https://cdn.waypoint.ai/v1.js"
  data-key="wpt_your_api_key"
  defer
></script>
// app/layout.tsx
import Script from 'next/script'

export default function RootLayout({ children }) {
  return (
    <html>
      <body>
        {children}
        <Script
          src="https://cdn.waypoint.ai/v1.js"
          data-key="wpt_your_api_key"
          strategy="afterInteractive"
        />
      </body>
    </html>
  )
}
# Add to your deploy pipeline (Vercel / Netlify / GitHub Actions)

# Option A: curl
curl -X POST https://api.waypoint.ai/reindex \
  -H "x-api-key: wpt_your_api_key"

# Option B: GitHub Actions step
- name: Trigger Waypoint reindex
  run: |
    curl -X POST https://api.waypoint.ai/reindex \
      -H "x-api-key: ${{ secrets.WAYPOINT_KEY }}"
